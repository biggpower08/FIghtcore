# Upgrade Paths

Current build scope: Ronin, Supreme Emperor, Monkey Grunt, and Monkey Striker.

The upgrade screen now surfaces a cyber warning that enemy strength rises slowly between waves. The current tuning is intentionally mild: enemy health can rise up to about 12%, while speed can rise up to about 6%. This keeps later waves more alert without turning the run into a hard-mode spike.

Existing upgrade paths remain the live design:

- Ronin: chain rhythm, density survival, counter-step activity, low-line control, and clean-cross pressure.
- Supreme Emperor: instant-death focus, royal pressure, crown breaker, imperial tempo, dominance armor, and finish sustain.
- Shared: activity gain, move recovery, health regeneration, dash cooldown, max activity, control, and wave recovery.

Do not implement larger upgrade trees until the core combat feel and sprite readability are stable.
